# QQ空间导出助手

> QQ空间导出助手，用于备份QQ空间的日志、说说、相册、留言板为文件，供永久保存

## 简介

QQ空间导出助手，用于备份QQ空间的日志、说说、相册、留言板为文件，供永久保存。

## 安装
- 下载源码
- 打开[谷歌扩展](chrome://extensions/)
- 勾选开发者模式
- 点击[加载已解压的扩展程序]按钮
- 选择QZoneExport/src文件夹


## 使用
- 登录QQ空间
- 右上角点击插件图标
- 弹窗点击开始备份
- 等待备份完成
- 点击下载备份
- 保存
